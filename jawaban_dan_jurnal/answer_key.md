## Rumus Penting

### PPh Badan
```
PKP = Penghasilan Neto Akuntansi ± Koreksi Fiskal
PPh = Tarif × PKP
Tarif 2024: ≤ Rp 4,8M = 11%, > Rp 4,8M = 22%
```

### PPh Orang Pribadi
```
PKP = Penghasilan Neto - PTKP
PTKP 2024: Diri = Rp 54Jt, Kawin = +Rp 4,5Jt, Anak = +Rp 4,5Jt (maks 3)
Tarif Progresif: 5%, 15%, 25%, 30%, 35%
```

### PPN
```
PPN = 11% × Dasar Pengenaan Pajak
PPN Keluaran = PPN atas penjualan
PPN Masukan = PPN atas pembelian
```

### PBB
```
PBB = 0,5% × NJKP
NJKP = Persentase NJKP × NJOP
NJKP: ≤ Rp 2M = 40%, > Rp 2M = 40-100%
```

### Bea Meterai
```
Nilai ≤ Rp 5.000.000 = Rp 0
Nilai > Rp 5.000.000 = Rp 10.000
```
